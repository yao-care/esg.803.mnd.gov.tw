# 利害關係人鑑別矩陣

document_id: MTX-STAKEHOLDER

## 1. 目的與範圍

本矩陣依據 GRI 2-29「利害關係人參與方式」標準，系統性鑑別國軍臺中總醫院之主要利害關係人，分析其關注議題、影響程度及本院之溝通回應機制，作為 ESG 管理決策與資訊揭露之基礎依據。

**更新週期：** 每年 Q1 由醫務企劃管理室更新，提交 ESG 委員會審查確認。

## 2. 利害關係人鑑別原則

利害關係人之鑑別依下列原則進行：

1. **依賴關係：** 對本院服務、決策或活動有所依賴的個人或群體。
2. **影響關係：** 可能影響或受本院活動影響的個人或群體。
3. **責任關係：** 本院對其負有法律、道義或財務責任之群體。

影響程度分級：
- **高：** 能直接影響本院營運決策，或受本院活動直接且重大影響。
- **中：** 間接影響本院營運，或受本院活動間接影響。
- **低：** 對本院活動有一般性關注，影響相對有限。

| **國防部軍醫局** | 醫療品質評核、溫室氣體盤查法規遵循、ESG 揭露要求、預算績效、軍事醫療支援能量 | 高 | 公文往來、業務報告、專案會議、稽核視察 | 每季/視需要 | 公文回覆（14 日內）、專案改善報告、年度業務評鑑自評 |
| **環境部** | 溫室氣體排放申報、碳費繳納合規、廢水排放標準、廢棄物清除申報 | 高 | 法規公告、排放量申報平臺、稽查通知 | 每年/視需要 | 盤查報告書申報（年度）、稽查配合、缺失改善回報 |
| **衛生福利部** | 醫院評鑑合格、感染管制、病人安全、醫療廢棄物處置 | 高 | 評鑑公告、書面評鑑報告、現場評鑑、函文 | 每三年（評鑑週期）/視需要 | 評鑑自評報告、改善計畫提交、缺失限期改正 |
| **病患及家屬** | 醫療品質、就醫安全、醫病溝通、環境舒適度、隱私保護 | 高 | 門診/住院服務、衛教說明、病患滿意度調查、申訴管道 | 每次就醫/每月 | 滿意度調查月報、申訴 24 小時受理、FRM-SATISFACTION 回饋追蹤 |
| **員工（含軍職/約聘）** | 職場安全衛生、薪資福利、職業發展、工作環境、ESG 政策認知 | 高 | 院內公告、單位主管傳達、教育訓練、職安衛委員會、員工意見箱 | 每月/視需要 | 職安衛委員會月會、教育訓練紀錄、員工意見 7 個工作天內回覆 |

| **供應商/承攬商** | 採購規範、付款時效、合約條款、ESG 要求（綠色採購）、勞動條件 | 中 | 採購公告、合約文件、廠商說明會、聯合採購小組 | 每次採購案/每年 | 採購爭議申訴管道、合約規範遵循確認、ESG 條款納入採購規格 |
| **查證機構** | 盤查數據品質、方法論適切性、查證程序配合、文件完整性 | 中 | 書面資料提交、現場查證作業、發現事項溝通 | 每年（查證期間） | 查證發現 7 個工作天內回覆、矯正措施報告提交（PRO-GHG-VERIFY） |

## 4. 重大性分析（雙重重大性）

依 GRI 3 重大主題識別程序，結合利害關係人關注度與本院實際衝擊，鑑別下列重大 ESG 議題：

| 議題 | 利害關係人關注度 | 本院實際衝擊 | 重大性 |
|------|:---:|:---:|:---:|
| 溫室氣體排放與減碳 | 高 | 高 | 重大 |
| 醫療品質與病人安全 | 高 | 高 | 重大 |
| 職業安全衛生 | 高 | 高 | 重大 |
| 醫療廢棄物管理 | 高 | 高 | 重大 |
| 能源管理效率 | 中 | 高 | 重大 |
| 社區健康促進 | 中 | 中 | 中度重大 |
| 綠色採購 | 中 | 中 | 中度重大 |
| 水資源管理 | 低 | 中 | 一般關注 |
| 建案環境影響 | 低 | 中 | 一般關注 |
