# 節能設備選用指引

document_id: GDL-EQUIP

## 1. 指引目的與範圍

本指引提供國軍臺中總醫院採購節能設備時之選用建議，協助採購人員、行政組及各單位主管，在設備汰換與新採購決策時有一致的能效評估標準，以落實 PLN-ENERGY 節能減碳行動計畫之目標。

本指引為**非強制性文件**，所列標準為建議門檻值，實際採購應以政府採購法規定之程序辦理。採購單位得依實際預算及設備規格，參考本指引要求作為技術規格訂定之依據。

本指引適用之設備類型：照明設備、空調設備（分離式冷氣機、冷水主機）、醫療輔助設備（含一般辦公設備）。

## 2. 適用對象

| 對象 | 使用時機 |
|------|------|
| 行政組採購承辦人 | 設備採購規格訂定 |
| 聯合採購小組 | 共同採購規格審查 |
| 各單位主管 | 提出設備需求申請時的能效說明 |
| 醫務企劃管理室 | ESG 績效追蹤時之標準依據 |

## 3. 指引內容

### 3.1 照明設備 — LED 燈具

#### 3.1.1 汰換判斷原則

建議優先汰換以下類型之既有照明：

- T8 傳統螢光燈管（使用年限超過 5 年者）
- 鹵素燈、白熾燈（任何年限，一律建議汰換）
- 金屬鹵化物燈（HID，使用年限超過 3 年者）

#### 3.1.2 LED 燈具建議規格

| 規格項目 | 建議值 | 說明 |
|------|------|------|
| 發光效率（Luminous Efficacy） | ≥ 100 lm/W | T8 LED 燈管；標準辦公照明 |
| 演色指數（CRI / Ra） | ≥ 80 | 病房、診間建議 ≥ 90 |
| 色溫 | 3000K～5000K | 依空間用途選擇；手術室宜 4000K 以上 |
| 功率因數（Power Factor） | ≥ 0.9 | 降低諧波污染 |
| 壽命 | ≥ 35,000 小時 | L70（光通量維持 70% 時之壽命） |
| 能源標章 | 1 級或 2 級 | 依經濟部能源署能源標章等級 |
| CNS 認證 | 應具備 | CNS 15630 系列或對應認證 |

#### 3.1.3 節電效益估算公式

**年節電量（度）=（舊燈具瓦數 - LED 瓦數）× 每日使用小時 × 365 / 1000**

| 變數 | 單位 | 說明 |
|------|------|------|
| 舊燈具瓦數 | W | 原有燈具標示功率（含安定器損耗） |
| LED 瓦數 | W | LED 燈具輸入功率 |
| 每日使用小時 | h/day | 該空間平均每日照明使用時數 |

**計算範例：** 以 T8 40W 傳統燈管（含安定器約 48W）換為 20W LED 燈管，每日使用 12 小時：

年節電量 = (48 - 20) × 12 × 365 / 1000 = **122.64 度/支/年**

### 3.2 空調設備 — 分離式冷氣機

#### 3.2.1 汰換判斷原則

建議優先汰換：

- 使用年限超過 10 年之舊型定頻機
- 能源標章等級低於 3 級之機種（或無能源標章）
- 使用 R-22 冷媒之舊機（符合環保法規淘汰期限）

#### 3.2.2 分離式冷氣機建議規格

| 規格項目 | 建議值 | 說明 |
|------|------|------|
| 冷氣能源效率比（EER / CSPF） | CSPF ≥ 4.5（變頻機） | 依 CNS 14464 測試標準 |
| 能源標章等級 | 1 級 | 年耗電量最省之等級 |
| 型式 | 變頻式（Inverter） | 較定頻機節電 20%~40% |
| 冷媒種類 | R-32 或 R-410A | 建議選用 R-32（GWP 較低：677 vs 1,923.5） |
| 噪音（室內機） | ≤ 40 dB(A) | 病房區宜 ≤ 35 dB(A) |

#### 3.2.3 能效比較計算

**年節電量（度）= 冷凍噸數 × 年操作小時 / EER（舊） - 冷凍噸數 × 年操作小時 / EER（新）**

**範例：** 2 噸分離式冷氣（舊機 EER=2.5，新機 CSPF=4.5），全年操作 2,000 小時：

- 舊機年耗電 = 2 × 3.517（kW/冷凍噸）× 2000 / 2.5 = **5,627 度**
- 新機年耗電 = 2 × 3.517 × 2000 / 4.5 = **3,126 度**
- **年節電 = 2,501 度**

### 3.3 冷水主機（Chiller）— 中央空調

#### 3.3.1 選用建議

| 規格項目 | 建議值 | 說明 |
|------|------|------|
| 整合部分負載值（IPLV） | ≥ 6.5 kW/RT | 依 AHRI 550/590 標準 |
| 型式 | 變頻螺旋式或離心式 | 大型冷量（≥200RT）優先考量離心 |
| 冷媒 | R-134a 或 R-1234ze(E) | 建議低 GWP 冷媒 |
| 熱回收功能 | 建議納入 | 可回收冷凝熱供熱水使用 |
| 能源監測介面 | Modbus RTU 或 BACnet | 供能源管理系統介接 |

#### 3.3.2 IPLV 節能計算

**年節電估算（度）= 年冷氣度日（CDDH）× 總冷凍噸 / IPLV**

| 變數 | 說明 |
|------|------|
| 年冷氣度日（CDDH） | 臺中地區約 2,400 小時/年（依 CNS 能效計算基準） |
| 總冷凍噸 | 主機額定冷凍噸數 |
| IPLV | kW/冷凍噸（越小越節能） |

### 3.4 一般辦公與醫療輔助設備

| 設備類型 | 建議要求 |
|------|------|
| 桌上型電腦/主機 | 具備 Energy Star 或相等認證；電源管理設定啟用自動休眠（閒置 15 分鐘後） |
| 印表機/多功能事務機 | Energy Star 認證；雙面列印設為預設 |
| 冰箱/冷凍庫（一般） | 能源標章 1 級 |
| 醫療冰箱（藥品/檢體） | 具能源標章或原廠能耗說明書；年耗電量明確標示 |
| UPS 不斷電系統 | 轉換效率 ≥ 90%（EC 模式） |

## 5. 附錄

### 5.1 設備採購能效審查檢核表

採購申請單位填寫設備需求時，建議檢核以下項目：

| 項目 | 應提供資料 | 符合 |
|------|------|:---:|
| 設備類型 | 明確標示（照明/空調/辦公設備等） | ☐ |
| 能效標章等級 | 附上能源標章或型錄影本 | ☐ |
| 預估年耗電量 | 依規格計算或廠商提供 | ☐ |
| 替換舊設備型號 | 說明汰換設備現況（使用年限、能效） | ☐ |
| 節電效益試算 | 依 3.1.3 或 3.2.3 公式計算 | ☐ |
| 冷媒種類（空調類） | 確認非受管制冷媒（R-22 禁止採購） | ☐ |

### 5.2 常見問題（Q&A）

**Q1：T8 LED 燈管與 T8 傳統燈管（磁力安定器）相容嗎？**
A：部分 LED 燈管宣稱相容，但建議同步汰換安定器或採用電子式驅動器，以確保 LED 燈管壽命及安全性。採購時確認廠商提供的相容性說明。

**Q2：採購冷氣機時，如何確認是否為真正變頻機而非假變頻？**
A：要求廠商提供 CSPF 測試報告（依 CNS 14464 之季節能源效率比），CSPF 數值應明確標示於設備銘板或型錄。真變頻機在部分負載下能效顯著優於定頻機。

**Q3：能源標章 1 級比 2 級節電多少？**
A：依能源署公告，各類設備能效等級門檻不同。以分離式冷氣為例，1 級 CSPF 約為 2 級之 1.1~1.2 倍，全年節電效益因機型而異，採購時宜直接比較年耗電量數值（標示於能源標章）。

**Q4：小型採購（金額低於公告金額）是否也需遵循本指引？**
A：本指引為建議性文件，小型採購仍建議以能效標準作為選用依據，以累積全院整體節能效益。
