# 排放源清冊

document_id: MTX-EMISSION

## 1. 矩陣目的與範圍

本排放源清冊系統性列出國軍臺中總醫院組織邊界內所有溫室氣體排放源，作為年度盤查作業之基礎文件。本清冊涵蓋類別1直接排放與類別2間接排放之所有排放源，包含排放形式、設施/活動、溫室氣體種類及量化方法。

**使用情境：**
- 年度盤查規劃時確認排放源範圍
- 數據蒐集責任分配之依據
- 內部/外部查證時之比對基準
- 新增或排除排放源之變更管理

## 2. 列/欄定義

| 欄位 | 說明 |
|------|------|
| 排放源 | 該排放活動的名稱 |
| 類別 | ISO 14064-1 類別（類別1 直接排放 / 類別2 間接排放） |
| 排放形式 | 固定源、移動源、逸散源、間接源 |
| 設施/活動 | 產生排放之具體設施或活動 |
| 溫室氣體種類 | 可能排放之溫室氣體（CO2、CH4、N2O、HFCs） |
| 量化方法 | 該排放源採用之量化計算方式 |
| 數據來源單位 | 活動數據之提供單位 |

本清冊之數據擷取自以下文件：

| 來源文件 | document_id | 擷取內容 |
|------|------|------|
| 2025年溫室氣體盤查報告書 | RPT-GHG-2025 | 排放源清單（表3.2節）、活動數據管理（表4.1節） |
| 2023年溫室氣體盤查報告書 | RPT-GHG-2023 | 報告邊界調查表（表2-1） |

## 5. 更新頻率

| 觸發條件 | 更新內容 | 負責人 |
|------|------|------|
| 每年度盤查規劃階段（Q1） | 全面檢視排放源增減 | 醫務企劃管理室副召集人 |
| 新增設備或製程 | 新增排放源並鑑別量化方法 | 醫務企劃管理室副召集人 |
| 設備報廢或停用 | 標記排放源為停用 | 醫務企劃管理室副召集人 |
| 排放係數或 GWP 版本更新 | 更新量化方法欄位 | 醫務企劃管理室副召集人 |
| 組織邊界變更 | 重新鑑別排放源 | 醫務企劃管理室副召集人 |
| 年度報告書（RPT）發行後 | 確認清冊與報告一致 | 醫務企劃管理室副召集人 |

## 6. 追溯性

本清冊中每筆排放源資料可透過以下路徑追溯至原始紀錄：

1. **排放源鑑別** → RPT-GHG-2025 第2章「報告邊界」
2. **活動數據** → 各 FRM 表單月報資料 → 原始憑證（發票、帳單、系統報表）
3. **排放係數** → GDL-CALC-METHOD 排放係數表 → 環境部公告/IPCC 文獻
4. **量化方法** → GDL-CALC-METHOD 量化公式
5. **數據品質** → RPT-GHG-2025 第4章「數據品質管理」

追溯鏈：FRM 原始憑證 → FRM 月報表 → MTX-EMISSION 清冊 → RPT-GHG 報告書
