# 綠建築設計指引

document_id: GDL-GREEN-BUILD

## 1. 目的與範圍

本指引提供國軍臺中總醫院新建及改建工程之綠建築設計參考原則，以取得內政部建築研究所「建築物節約能源設計技術規範（EEWH）」認證為目標，確保建築物在生命週期內降低能源消耗、減少環境衝擊，並符合 PLN-CONSTRUCTION 建案 ESG 規劃之環境目標。

**適用對象：** 本院建案設計單位、施工承攬商、行政組建案管理人員。

**適用範圍：** 新建醫療大樓及樓地板面積 500 平方公尺以上之改建工程。

本指引為非強制性參考文件，具體設計規格以各建案設計圖說及採購規範為準，本指引提供評選與審查方向。

## 2. EEWH 綠建築標章制度

### 2.1 認證等級

內政部 EEWH（Ecology、Energy、Waste Reduction、Health）綠建築標章共分五個等級：

| 等級 | 名稱 | 評分門檻 | 適合目標 |
|------|------|:---:|------|
| ★ | 合格級 | 通過基本要求 | 法規最低合規要求 |
| ★★ | 銅級 | 得分 ≥ 1 | 超越法規基本要求 |
| ★★★ | 銀級 | 得分 ≥ 2 | 中等綠建築表現 |
| ★★★★ | 黃金級 | 得分 ≥ 3 | 優良綠建築表現 |
| ★★★★★ | 鑽石級 | 得分 ≥ 4 | 頂尖綠建築表現 |

**本院目標：** 新建醫療大樓取得銀級以上認證（依 PLN-CONSTRUCTION G2-01）。

### 2.2 九大評估指標概覽

EEWH 評估系統涵蓋九大指標，分為必要指標（必達）與選用指標（計分）：

| 類別 | 指標編號 | 指標名稱 | 類型 |
|------|:---:|------|:---:|
| 生態（Ecology） | 1 | 生物多樣性 | 選用 |
| | 2 | 綠化量 | 選用 |
| | 3 | 基地保水 | 選用 |
| 能源（Energy） | 4 | 日常節能 | **必要** |
| 減廢（Waste Reduction） | 5 | CO₂ 減量 | 選用 |
| | 6 | 廢棄物減量 | 選用 |
| 健康（Health） | 7 | 室內環境 | **必要** |
| | 8 | 水資源 | 選用 |
| | 9 | 污水垃圾改善 | **必要** |

## 3. 能源效率設計原則

### 3.1 建築外殼節能

| 項目 | 設計建議 | 說明 |
|------|------|------|
| 外牆隔熱 | 外牆整體熱傳透率（U 值）≤ 1.5 W/m²K | 可採用隔熱砂漿、RC 複合隔熱板或通風外牆 |
| 屋頂隔熱 | 屋頂整體熱傳透率（U 值）≤ 1.0 W/m²K | 可採用隔熱磚、綠屋頂或反射塗料 |
| 外遮陽設計 | 南向立面設置固定式水平遮陽板，東西向設置垂直遮陽板 | 降低日射得熱，減少空調負荷 |
| 窗戶玻璃 | 採用低輻射（Low-E）玻璃，遮陽係數（SC）≤ 0.5 | 隔熱兼顧採光 |
| 窗牆比 | 一般辦公區域窗牆比控制在 20–40% | 平衡採光與隔熱需求 |

### 3.2 空調系統

| 項目 | 設計建議 | 說明 |
|------|------|------|
| 主要冷源設備 | COP（成績係數）≥ 5.0（採用高效率冰水主機） | 選用能效 1 級設備 |
| 分離式冷氣 | COP ≥ 3.8（相當於能效 1 級） | 局部空調使用 |
| 變頻控制 | 冰水主機、送風機、冷卻水泵全數採用變頻控制 | 依負荷動態調節，大幅節電 |
| 新鮮外氣引入 | 外氣量依 CNS 14816 標準，設定 CO₂ 濃度回饋控制 | 兼顧空氣品質與節能 |
| 廢熱回收 | 評估全熱交換器導入，回收排氣廢熱預冷（預熱）外氣 | 可減少 20–30% 新鮮空氣空調負載 |

### 3.3 照明系統

| 項目 | 設計建議 | 說明 |
|------|------|------|
| 光源選擇 | 全面採用 LED 光源，色溫 3000–4000K（病房）、4000–5000K（診間/辦公） | 節能效率較傳統燈管提升 50% 以上 |
| 照明密度 | 符合 CNS 12112 照明標準，各空間照度不超標設計 | 辦公區 ≤ 10 W/m²，病房 ≤ 7 W/m² |
| 照明控制 | 公共區域設置人體感測器自動控制；日光充足區域設置日光感測調光 | 減少不必要用電 |
| 緊急照明 | 全數採用 LED 緊急燈具，電池容量符合 90 分鐘連續使用要求 | — |

## 4. 室內環境品質（IEQ）

### 4.1 室內空氣品質

| 項目 | 標準 | 依據 |
|------|------|------|
| CO₂ 濃度 | 一般空間 ≤ 1,000 ppm | 室內空氣品質管理法 |
| 甲醛 | ≤ 0.08 ppm | 室內空氣品質管理法 |
| 總揮發性有機化合物（TVOC） | ≤ 0.56 ppm | 室內空氣品質管理法 |
| 換氣次數 | 一般病房 ≥ 6 次/小時，手術室 ≥ 20 次/小時 | CNS 14816 |
| 正壓/負壓控制 | 隔離病房設置負壓控制（相對於走廊） | 感染管制需求 |

### 4.2 建材選擇

| 項目 | 設計建議 | 說明 |
|------|------|------|
| 綠建材標章 | 採用內政部核定綠建材，目標占室內總面積 ≥ 60% | 低甲醛、低 VOC 建材 |
| 木材來源 | 若使用木材，優先採用 FSC 認證木材 | 確保合法永續來源 |
| 油漆塗料 | 採用水性低 VOC 塗料（VOC 含量 ≤ 50 g/L） | 降低施工期及使用期空氣污染 |

## 5. 水資源效率

| 項目 | 設計建議 | 說明 |
|------|------|------|
| 節水器具 | 全面採用省水標章器具（省水率 ≥ 20%） | 廁所馬桶、洗手台、沖洗水龍頭 |
| 雨水回收 | 設置屋頂雨水收集槽，回收用於廁所沖水及植栽澆灌 | 可降低自來水用量 20–30% |
| 廢水分質回收 | 評估空調冷卻水塔排水回收再利用可行性 | 適用冷卻水塔用水量大之建物 |
| 防漏偵測 | 設置用水量計量系統，各樓層獨立計量，異常用水自動警示 | 及早偵測管路漏水 |

## 6. 施工廢棄物減量

| 措施 | 說明 |
|------|------|
| 廢棄物分類設置 | 施工現場設置廢混凝土、廢木材、廢金屬、廢紙/紙板、廢塑膠、廢油漆容器分類區 |
| 資源化再利用 | 廢鋼筋、廢金屬委託合法資源回收商；廢混凝土委託再生粒料廠處理 |
| 廢棄物月報 | 承攬商每月提交廢棄物清除申報紀錄（廢棄物種類、數量、清除機構） |
| 目標資源化率 | 施工廢棄物資源化率 ≥ 70% |
