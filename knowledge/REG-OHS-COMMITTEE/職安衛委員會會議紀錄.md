# 職安衛委員會會議紀錄

document_id: REG-OHS-COMMITTEE

## 1. 紀錄目的與範圍

本文件規範國軍臺中總醫院職業安全衛生委員會（以下簡稱職安衛委員會）會議紀錄之格式、填寫規範及保存要求，確保委員會決議事項得以完整追蹤與執行。

**記錄對象：** 每次職安衛委員會正式會議。

**適用範圍：** 職業安全衛生室主辦之各次定期委員會會議，包含例行季會及臨時召開之緊急會議。

**上位程序：** PRO-OHS（職業安全衛生管理程序）

**與評鑑關聯：** 本紀錄為 STD-ACCRED 第二章（人力資源管理）評鑑佐證文件之一，評鑑委員實地訪查時須備齊最近 3 年會議紀錄。

## 2. 紀錄格式

### 2.1 會議基本資訊

| 欄位 | 資料型態 | 說明 | 必填/選填 |
|------|------|------|------|
| 會議編號 | 文字 | 格式：OHS-YYYY-QQ（如 OHS-2026-Q1） | 必填 |
| 會議日期 | 日期 | 格式：YYYY-MM-DD | 必填 |
| 會議時間 | 時間 | 格式：HH:MM ~ HH:MM | 必填 |
| 會議地點 | 文字 | 具體地點（如「行政大樓 3 樓會議室」） | 必填 |
| 主席 | 文字 | 職稱（如「院長」或「副院長」） | 必填 |
| 記錄人 | 文字 | 職稱及姓名 | 必填 |
| 列席人數 | 數字 | 出席委員總人數 | 必填 |

### 2.2 出席人員

| 欄位 | 資料型態 | 說明 | 必填/選填 |
|------|------|------|------|
| 單位 | 文字 | 所屬單位名稱 | 必填 |
| 職稱 | 文字 | 委員職稱 | 必填 |
| 出席狀態 | 選擇 | 出席／請假（請假須填代理人） | 必填 |
| 代理人 | 文字 | 代理出席者姓名及職稱（請假時填寫） | 選填 |

### 2.3 議程與決議

| 欄位 | 資料型態 | 說明 | 必填/選填 |
|------|------|------|------|
| 議案編號 | 文字 | 格式：OHS-YYYY-QQ-NN（如 OHS-2026-Q1-01） | 必填 |
| 議案名稱 | 文字 | 議案摘要標題 | 必填 |
| 提案單位 | 文字 | 提出議案之單位 | 必填 |
| 說明 | 長文字 | 議案背景、現況及建議事項 | 必填 |
| 決議 | 長文字 | 委員會決議內容，含贊成/反對/修正意見 | 必填 |
| 執行單位 | 文字 | 負責執行決議之單位 | 必填 |
| 預定完成日 | 日期 | 格式：YYYY-MM-DD | 必填 |

### 2.4 追蹤事項（上次決議追蹤）

| 欄位 | 資料型態 | 說明 | 必填/選填 |
|------|------|------|------|
| 原議案編號 | 文字 | 參照上次會議議案編號 | 必填 |
| 執行狀態 | 選擇 | 已完成／執行中／逾期／擱置 | 必填 |
| 完成日期 | 日期 | 實際完成日期（已完成時填寫） | 選填 |
| 執行說明 | 長文字 | 執行進度或說明 | 必填 |

## 3. 填寫規範

- 會議結束後 5 個工作天內完成紀錄初稿，送主席確認。
- 主席確認後 3 個工作天內完成定稿，並分發所有出席委員及主席。
- 議案決議欄須明確記載決議結論，不得以「參考辦理」等模糊字句帶過。
- 出席人員以職稱記錄，不得僅填寫姓名而未附職稱。
- 請假委員若無代理人出席，應於說明欄記載請假原因。
- 上次決議追蹤須逐案回報，執行中之議案須說明進度百分比或預計完成時程。
- 紀錄完成後由記錄人及主席親筆簽名後，正本存於職業安全衛生室，副本由醫務企劃管理室留存一份（供評鑑備查）。

## 4. 更新頻率

職安衛委員會每季召開一次例行會議，固定排於每季第二個月中旬（建議日期：2月、5月、8月、11月）。

| 更新觸發條件 | 說明 |
|------|------|
| 例行季會 | 每季一次，不得連續兩季未召開 |
| 重大職災發生 | 嚴重職災（導致死亡、住院或重大財物損失）後 10 個工作天內召開臨時會 |
| 法規重大修訂 | 職業安全衛生相關法規重大修正時，視必要性召開臨時會 |
| 評鑑前審查 | 評鑑年度之評鑑前 3 個月加開一次，專項審查合規狀況 |

## 5. 保存與調閱

| 項目 | 說明 |
|------|------|
| 保存期限 | 7 年 |
| 正本存放位置 | 職業安全衛生室（實體文件櫃） |
| 電子備份位置 | 醫院共享磁碟機 / 職安衛資料夾 |
| 調閱權限 | 職業安全衛生室全體人員、醫務企劃管理室主任、院長（稽核用途）；評鑑期間開放評鑑委員調閱 |
| 銷毀程序 | 保存期限屆滿後，由職業安全衛生室主任核定，以碎紙機銷毀實體文件，並於電子系統刪除電子檔案 |
