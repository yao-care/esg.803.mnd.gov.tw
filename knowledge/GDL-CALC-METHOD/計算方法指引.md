# 溫室氣體計算方法指引

document_id: GDL-CALC-METHOD

## 1. 指引目的與範圍

本指引提供國軍臺中總醫院溫室氣體盤查作業所需之量化計算方法、排放係數、全球暖化潛勢值（GWP）及不確定性評估方法，作為各排放源計算之參考依據。

本指引為非強制性文件，旨在提供一致且正確的計算方法參考，各權責單位可依據本指引之公式與係數進行排放量計算。若排放係數或 GWP 值因法規更新而變動，應以最新公告版本為準。

## 2. 適用對象

- 醫務企劃管理室：負責排放量彙算與報告書編製
- 行政組：油料、電力、冷媒等活動數據之計算
- 醫勤組：化糞池排放之計算
- 衛材補給保養室：氣體鋼瓶、醫用冷媒之計算
- 碳盤查小組全體成員：查證時之計算複核

## 3. 指引內容

### 3.1 基本量化公式

溫室氣體排放量之基本計算公式如下：

**CO2e = 活動數據 x 排放係數 x GWP**

| 變數 | 定義 | 單位 |
|------|------|------|
| CO2e | 二氧化碳當量排放量 | 公噸 CO2e |
| 活動數據 | 排放活動之使用量或發生量 | 依排放源而異 |
| 排放係數 | 單位活動數據之溫室氣體排放量 | 依排放源而異 |
| GWP | 全球暖化潛勢值 | 無因次 |

### 3.2 各排放源量化公式

#### 3.2.1 固定源/移動源燃料燃燒

排放量（tCO2e）= 燃料購買量（公秉）x [CO2排放係數 x CO2 GWP + CH4排放係數 x CH4 GWP + N2O排放係數 x N2O GWP]

#### 3.2.2 車用尿素水解

排放量（tCO2e）= 購買量（L）x 比重（1.09 kg/L）x 尿素濃度（17.5%）/ 尿素分子量（60）x CO2分子量（44）/ 1000

化學反應式：CO(NH2)2 + H2O → 2NH3 + CO2

#### 3.2.3 化糞池甲烷逸散

排放量（tCO2e）= 人員在院時間（人時）x CH4排放係數（公噸/人時）x CH4 GWP

#### 3.2.4 冷媒逸散（補充量法）

排放量（tCO2e）= 冷媒補充量（公噸）x 排放係數（1.0 公噸/公噸）x 冷媒GWP

冷媒補充量 = 新購設備冷媒填充量 + 既有設備維修冷媒補充量

#### 3.2.5 麻醉氣體逸散

排放量（tCO2e）= 麻醉藥品使用量（公噸）x GWP

#### 3.2.6 氣體鋼瓶逸散

排放量（tCO2e）= 撥發量（支）x 單位填充量（kg/支）/ 1000 x GWP

#### 3.2.7 滅火器逸散

排放量（tCO2e）= 購買量（支）x 單位填充量（kg/支）/ 1000 x GWP

#### 3.2.8 外購電力

排放量（tCO2e）= 淨用電度數（千度）x 電力排碳係數（公噸 CO2e/千度）

淨用電度數 = 電費帳單度數 - 租用單位抄表度數

### 3.7 不確定性計算方法

#### 3.7.1 計算公式

1. **活動數據不確定性：** 活動數據不確定性上下限 = ± 誤差值（%）x 擴充係數（2）
2. **單一溫室氣體不確定性：** √（活動數據不確定性² + 排放係數不確定性²）
3. **單一排放源不確定性：** √（Σ（各氣體不確定性 x 排放當量）²）/ Σ各氣體排放當量
4. **總不確定性：** √（Σ（各排放源不確定性 x 排放當量）²）/ Σ排放當量

## 5. 附錄

### 5.1 計算範例：公務車汽油排放

**已知條件：**
- 2025年度車用汽油購買量：5.734 公秉
- CO2 排放係數：2.2631328720 公噸/公秉
- CH4 排放係數：0.0008164260 公噸/公秉
- N2O 排放係數：0.0002612563 公噸/公秉
- GWP（AR6）：CO2=1, CH4=27.9, N2O=273

**計算過程：**
- CO2 排放量 = 5.734 x 2.2631328720 x 1 = 12.9746 tCO2e
- CH4 排放量 = 5.734 x 0.0008164260 x 27.9 = 0.1306 tCO2e
- N2O 排放量 = 5.734 x 0.0002612563 x 273 = 0.4089 tCO2e
- **合計 = 13.5141 tCO2e**

### 5.2 計算範例：化糞池排放（員工）

**已知條件：**
- 2025年度員工在院人時：1,684,124.5 人時
- CH4 排放係數：0.0000015938 公噸/人時
- CH4 GWP（AR6）：27.9

**計算過程：**
- CH4 排放量 = 1,684,124.5 x 0.0000015938 x 27.9 = 74.8638 tCO2e

### 5.3 計算範例：外購電力排放

**已知條件：**
- 2025年度淨用電量：6,302.75568 千度（MWh）
- 113年電力排碳係數：0.474 公噸CO2e/千度

**計算過程：**
- 排放量 = 6,302.75568 x 0.474 = 2,987.5062 tCO2e

### 5.4 常見問題（Q&A）

**Q1：排放係數版本更新時如何處理？**
A：排放係數以計算當年度最新公告版本為準。若年度中途係數更新，應以報告書定版時之最新版本重新計算全年數據。

**Q2：冷媒種類不在七大管制氣體內（如 R-600a、R-290）如何處理？**
A：R-600a、R-170、R-290、R-1234yf 非七大管制溫室氣體，進行排放源鑑別但不予量化，於報告書中說明排除原因。

**Q3：GWP 值應引用 AR5 或 AR6？**
A：依外部查證機構與國家方案要求為準。2023年度盤查適用 AR5，2025年度盤查適用 AR6。

**Q4：電力排碳係數的年度如何對應？**
A：電力排碳係數以盤查年度前一年公告之最新係數為準。例如 2025 年度盤查使用 113 年（2024年）公告之電力排碳係數。
