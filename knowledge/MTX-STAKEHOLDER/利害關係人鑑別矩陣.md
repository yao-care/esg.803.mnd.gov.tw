# 利害關係人鑑別矩陣

document_id: MTX-STAKEHOLDER

## 1. 目的與範圍

本矩陣依據 GRI 2-29「利害關係人參與方式」標準，系統性鑑別國軍臺中總醫院之主要利害關係人，分析其關注議題、影響程度及本院之溝通回應機制，作為 ESG 管理決策與資訊揭露之基礎依據。

**更新週期：** 每年 Q1 由醫務企劃管理室更新，提交 ESG 委員會審查確認。

## 2. 利害關係人鑑別原則

利害關係人之鑑別依下列原則進行：

1. **依賴關係：** 對本院服務、決策或活動有所依賴的個人或群體。
2. **影響關係：** 可能影響或受本院活動影響的個人或群體。
3. **責任關係：** 本院對其負有法律、道義或財務責任之群體。

影響程度分級：
- **高：** 能直接影響本院營運決策，或受本院活動直接且重大影響。
- **中：** 間接影響本院營運，或受本院活動間接影響。
- **低：** 對本院活動有一般性關注，影響相對有限。

## 4. 重大性分析（雙重重大性）

依 GRI 3 重大主題識別程序，結合利害關係人關注度與本院實際衝擊，鑑別下列重大 ESG 議題：

| 議題 | 利害關係人關注度 | 本院實際衝擊 | 重大性 |
|------|:---:|:---:|:---:|
| 溫室氣體排放與減碳 | 高 | 高 | 重大 |
| 醫療品質與病人安全 | 高 | 高 | 重大 |
| 職業安全衛生 | 高 | 高 | 重大 |
| 醫療廢棄物管理 | 高 | 高 | 重大 |
| 能源管理效率 | 中 | 高 | 重大 |
| 社區健康促進 | 中 | 中 | 中度重大 |
| 綠色採購 | 中 | 中 | 中度重大 |
| 水資源管理 | 低 | 中 | 一般關注 |
| 建案環境影響 | 低 | 中 | 一般關注 |
