# 建案里程碑追蹤表

document_id: FRM-MILESTONE

## 1. 表單說明

本表單用於追蹤國軍臺中總醫院新建醫療大樓（及其他建案）之關鍵里程碑執行狀態。行政組承辦人每月更新一次，確認各里程碑之預定時程與實際執行情形，並於出現延遲時填寫說明與因應措施。

**填報週期：** 每月最後一個工作天

**填報單位：** 行政組

**審查人：** 行政組組長

**存查：** 醫務企劃管理室（ESG 委員會報告資料）

## 2. 填報欄位說明

| 欄位 | 說明 | 必填 | 格式 |
|------|------|:---:|------|
| 里程碑名稱 | 該里程碑之名稱（如：基礎工程完工、結構體完工、機電完工） | 是 | 文字 |
| 預定日期 | 原計畫預定完成日期 | 是 | YYYY-MM-DD |
| 實際日期 | 實際完成日期（未完成者留空） | 否 | YYYY-MM-DD |
| 狀態 | 目前執行狀態 | 是 | 未開始 / 進行中 / 完成 / 延遲 |
| 備註 | 延遲原因說明、風險說明或其他補充 | 否 | 文字 |

**狀態定義：**

- **未開始：** 尚未進入施工或作業
- **進行中：** 已開始但尚未完成
- **完成：** 已達到里程碑要求並驗收
- **延遲：** 實際進度落後預定日期超過 14 個工作天

## 4. 延遲處理程序

當任一里程碑狀態標記為「延遲」時，行政組承辦人應：

1. 於備註欄說明延遲原因（天氣因素、承攬商人力、採購時程等）。
2. 評估延遲對後續里程碑之連鎖影響。
3. 提出因應措施（如調整工序、追趕進度計畫）。
4. 若延遲影響建物點交日期（進而影響 ESG 盤查邊界擴充時程），應立即通知醫務企劃管理室，並更新 MTX-TIMELINE。
